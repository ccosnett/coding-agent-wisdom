# Intention

This project is a simple attempt to gather practical wisdom about using coding agents.

It begins with the writing of Peter Steinberger at `https://steipete.me/`.

The aim is not to create noise.
The aim is to read carefully, extract what is useful, and distill it into a clear `README.md`.

This project should become:

- A concise guide to using coding agents well
- A faithful distillation of ideas, not a careless summary
- A practical reference for real work
- A piece of writing with plain language and no hype

This project should avoid:

- Grand claims
- Vague advice
- Repetition
- Notes that do not help a person work better

Standard for the writing:

- Simple
- Accurate
- Useful
- Respectful of the source
- Professional in tone
